# 🧠 RH Disciplinar Analytics

Sistema genérico de análise de medidas disciplinares para geração automática de indicadores estratégicos de RH.

---

## 🎯 Objetivo

Transformar dados operacionais de medidas disciplinares em **informações estratégicas**, permitindo:

- Identificação de reincidência de colaboradores
- Análise de risco disciplinar
- Monitoramento de comportamento organizacional
- Apoio à tomada de decisão do RH

---

## 🚀 Funcionalidades

- 📂 Importação de planilhas Excel (.xlsx)
- 📊 Geração automática de KPIs:
  - Total de medidas
  - Reincidência (Sequência > 1)
  - Severidade média
  - Colaboradores impactados
- 📈 Análise temporal (evolução mensal)
- 🧠 Classificação automática de ocorrências (baseada em regras)
- 🏢 Distribuição por:
  - Departamento
  - Centro de custo
- 👥 Ranking de colaboradores com maior risco disciplinar
- 🔍 Busca e filtros dinâmicos
- 📋 Drill-down completo (histórico por colaborador)
- 📤 Exportação do dashboard completo em HTML

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (UI moderna e responsiva)
- JavaScript (Vanilla)
- XLSX.js (leitura de Excel no navegador)
- SVG (gráficos customizados)

---

## 🧩 Diferenciais

- ⚡ Processamento 100% local (sem necessidade de backend)
- 🔒 Nenhum dado é enviado para servidores externos
- 🎯 Foco em tomada de decisão (não apenas visualização)
- 🧠 Classificação automática de dados não estruturados
- 📊 Interface executiva pronta para apresentação

## ▶️ Como usar

1. Abra o arquivo `index.html` no navegador  
2. Importe uma planilha Excel (.xlsx ou .xls)  
3. Visualize automaticamente os KPIs e análises
